Commandes de démarrage
À lancer dans un terminal depuis le dossier racine
Frontend
# Terminal 1
cd frontend
pnpm dev
# → http://localhost:3002
Backend Node.js
# Terminal 2
cd backend-node
pnpm install
pnpm dev
# → http://localhost:4000
Backend Java
# Terminal 3
cd backend
./mvnw spring-boot:run
# → http://localhost:8080
Base de données
# Depuis frontend/
pnpm db:migrate
pnpm db:seed
pnpm db:studio
# → Prisma Studio :5555
